Create a Swarm on Discord#

This examples walks you through creating a swarm on Discord

Project Setup#

In your project: pip install osnap_client

Creating an application#

Follow the steps here to create a new discord application: https://discord.com/developers/docs/getting-started

Generate a Token#

Generate a token by navigating to Bot > Reset Token, or using an existing token. Run cp .env.template .env.local to copy the template environment file Add your Bot’s token as the SWARM_TOKEN in the .env.local file
